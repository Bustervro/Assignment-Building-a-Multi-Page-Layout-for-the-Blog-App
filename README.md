# Blog React Router App

This is a simple multi-page blog app built with React Router.

## Features

- Home feed page
- About page
- Dynamic post pages
- Persistent navigation bar
- File-based routing
- Outlet layout pattern
- useParams for dynamic post IDs
- useNavigate for returning to the feed

## How to Run

```bash
npm install
npm run dev
